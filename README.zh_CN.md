# Deepin Sound Theme

Deepin的自由桌面声音主题。

## 安装

```
make install
```

## 获取帮助

任何使用问题都可以通过以下方式寻求帮助：

* [Gitter](https://gitter.im/orgs/linuxdeepin/rooms)
* [IRC channel](https://webchat.freenode.net/?channels=deepin)
* [Forum](https://bbs.deepin.org/)
* [WiKi](http://wiki.deepin.org/)
